# Online Bookstore App

### Version

0.1.0

### About The Project

This project is created for an online bookstore web application. The frontend for the mentioned project was developed using

- Java Script
- HTML
- Sass Stylesheets
- React
- React Bootstrap
- Chakra UI
- Font Awesome Icons
- Google Fonts
- Supported by a REST API created using java
  <br>
